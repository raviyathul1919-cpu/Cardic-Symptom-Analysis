# Cardiac Symptom Analysis & Risk Prediction

## Project Overview
This academic capstone project focuses on evaluating large-scale healthcare datasets to analyze and predict cardiac risks. The system utilizes data profiling, extensive exploratory data analysis (EDA), and machine learning pipelines to turn raw medical observations into structured clinical intelligence.

## Tech Stack & Core Libraries
* **Programming Language:** Python
* **Data Processing & Analytics:** Pandas, NumPy
* **Machine Learning Frameworks:** Scikit-Learn
* **Database Management:** SQL
* **Web Architecture & Dashboarding:** Flask / Web Templates

## Data Methodology & Technical Workflow
1. **Data Ingestion & Cleaning:** Processed a high-volume healthcare dataset (`cardia.csv`) by managing database schemas (`cardia_flk.sql`) and handling null values to guarantee 100% data integrity.
2. **Exploratory Data Analysis (EDA):** Evaluated critical patient metrics and computed statistical correlations to map symptoms directly to cardiovascular risks.
3. **Model Training & Analytical QA:** Implemented and compared multiple predictive machine learning algorithms (including Logistic Regression, Naive Bayes, Random Forest, Decision Tree, KNN, and SVM).
4. **Performance Evaluation:** Generated and visualized rigorous validation outputs, including confusion matrices and model accuracy charts, to verify diagnostic precision.
5. **Deployment:** Structured a comprehensive web-based analytical dashboard interface using secure backend routes to display predictive risk insights seamlessly.
